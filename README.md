## 🎯 Возможности

- 🎨 **Красивый градиентный интерфейс** - логотип с переходом от красного к синему
- 🔧 **Смена MAC-адреса** - вручную или автоматическая генерация
- 💻 **Смена имени компьютера** - с валидацией и автогенерацией
- 🔄 **Управление адаптерами** - включение/отключение сетевых адаптеров
- 🤖 **Автоматическая смена при потере соединения** - мониторинг подключения к интернету
- 📊 **Интерактивное меню** - навигация стрелками (с библиотекой inquirer)
- ✅ **Валидация данных** - проверка корректности MAC-адресов и имен
- 🎲 **Умная генерация** - локально администрируемые MAC-адреса

## 🚀 Быстрый старт

### Требования

- Windows 10/11
- Python 3.7 или выше
- Права администратора

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/chelovechekss/netmorph.git
cd netmorph
```

2. Установите зависимости:
```bash
pip install inquirer
```

3. Запустите программу от имени администратора:
```bash
python macchanger.py
```

## 📖 Использование

### Основное меню

После запуска вы увидите интерактивное меню с опциями:

1. **Сменить MAC (вручную)** - введите свой MAC-адрес в формате `XX:XX:XX:XX:XX:XX`
2. **Сменить MAC (случайный)** - автоматическая генерация валидного MAC
3. **Сменить имя ПК (вручную)** - введите новое имя (3-15 символов, латиница, цифры, дефис)
4. **Сменить имя ПК (случайное)** - генерация имени вида `PC-XXXXXX`
5. **Включить/Отключить адаптер** - переключение состояния сетевого адаптера
6. **Автосмена при потере соединения** - запуск фонового мониторинга
7. **Показать текущие данные** - отображение текущего MAC и имени ПК
8. **Выбрать другой адаптер** - переключение между сетевыми адаптерами
9. **Выход** - завершение работы программы

### Автоматическая смена при потере соединения

Эта функция запускает фоновый поток, который:
- Пингует `8.8.8.8` каждую секунду
- При потере соединения на 5+ секунд автоматически меняет MAC и имя ПК
- Работает в фоне, не блокируя основное меню
- Можно включить/выключить в любой момент

**Использование:**
1. Выберите пункт "Автосмена при потере соединения"
2. Мониторинг запустится в фоновом режиме
3. При потере интернета программа автоматически сменит MAC и имя
4. Для остановки выберите этот пункт снова

## 🎨 Особенности интерфейса

### Цветовая схема

- `[*]` - Информационные сообщения (синий)
- `[E]` - Ошибки (красный)
- `[S]` - Успешные операции (зеленый)
- `[!]` - Предупреждения (желтый)

### Градиентный логотип

Логотип использует плавный переход от красного к синему цвету, создавая эффектный визуальный стиль.

## ⚙️ Технические детали

### Генерация MAC-адресов

Программа генерирует **локально администрируемые** MAC-адреса:
- Второй символ первого байта: `2`, `6`, `A` или `E`
- Пример: `A2:1B:3C:4D:5E:6F`
- Соответствует стандартам IEEE 802

### Работа с реестром

Программа автоматически:
- Находит адаптер в реестре Windows
- Записывает новый MAC в `NetworkAddress`
- Обновляет имя компьютера в нескольких ключах реестра
- Перезапускает адаптер для применения изменений

### Мониторинг соединения

- Использует многопоточность (`threading`)
- Проверяет доступность через `ping`
- Не блокирует основной интерфейс
- Автоматически останавливается при выходе

## 🛠️ Устранение неполадок

### Адаптер не найден в реестре

Программа пытается найти адаптер несколькими способами:
1. Через `wmic` - получение реального имени драйвера
2. Через PowerShell - альтернативный метод
3. По ключевым словам (Wi-Fi, Ethernet, и т.д.)

Если адаптер не найден, программа покажет список похожих адаптеров для выбора.

### MAC не изменился

Возможные причины:
- Драйвер адаптера не поддерживает смену MAC
- Недостаточно прав (запустите от администратора)
- Адаптер заблокирован антивирусом

### Требуется перезагрузка

После смены имени компьютера требуется перезагрузка для полного применения изменений.
